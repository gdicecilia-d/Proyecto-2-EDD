# Proyecto 2: Colas de Prioridad y HashTables

Este proyecto es una herramienta diseñada para la administración de recursos de impresión mediante el uso de colas de prioridad. El sistema permite gestionar el flujo de documentos basándose en tiempos
de llegada y rangos de usuario, utilizando un Montículo Binario (Min-Heap) para el ordenamiento eficiente y una Tabla de Dispersión (HashTable) para garantizar el acceso y eliminación de registros en 
tiempo constante O(1) sin que la cola conozca al propietario.
